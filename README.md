# 🚀 CryptoVerse - Neo Crypto Portfolio Tracker

![version](https://img.shields.io/badge/version-1.0-blue)
![license](https://img.shields.io/badge/license-MIT-green)

> Real-time cryptocurrency portfolio tracker dengan desain **neon futuristic** dan **glassmorphism**. Data live dari CoinGecko API.

## ✨ Fitur
- 💹 Harga real-time (BTC, ETH, SOL, ADA, XRP, DOGE)
- 💰 Total nilai portofolio (USD + IDR)
- 📈 Indikator profit/loss 24 jam
- 🔄 Auto refresh setiap 60 detik
- 📱 Responsive design

## 🔧 Cara Edit
Ubah jumlah holding kamu di variabel `myAssets` pada file `index.html`
